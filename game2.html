게임 규칙
① 식 만들기
  • 수 카드, 연산 카드(+,−,×,÷), 괄호 카드를 눌러(또는 끌어다) 목표 숫자를 만드는 식을 완성하세요.
  • 괄호는 안쪽부터 ( ) → { } → [ ] 순서로만 사용할 수 있어요.
  • 수 카드는 한 번 쓰면 식을 지울 때까지 다시 쓸 수 없어요.
② 점수 계산 (라운드당)
  • 다양성 점수: 서로 다른 정답 식을 찾을 때마다 (첫 정답 100점, 새 정답마다 25점씩 추가)
  • 정확도 보너스: 제출한 것 중 정답 비율 (최대 100점)
  • 시간 보너스: 라운드가 끝날 때 남아있는 시간 비율 (최대 100점)
③ 시간 보너스 팁
  • 더 찾을 답이 없다고 생각되면 시간이 남았을 때 🏁 라운드 종료 버튼을 눌러보세요. 남은 시간이 많을수록 보너스 점수가 커져요!
  • 힌트를 사용하면 점수에서 30점이 깎여요.
